# L 桥梁工具 - 快速开始指南

## 第一步：启动工具
- 双击 `start_L.bat` 启动工具
- 首次运行会自动安装依赖，可能需要一点时间

## 第二步：首次使用向导
- 打开后会显示「首次使用向导」
- 按照向导步骤采集模板：
  1. 选择项目配置
  2. 采集编程工具输入框
  3. 采集发送按钮（可跳过）
  4. 采集编程工具复制结果按钮
  5. 采集 ChatGPT 输入框
  6. 采集 ChatGPT 发送按钮（可跳过）
  7. 执行安全演练

## 第三步：安全检查
- 保持「安全模式」开启
- 点击「一键安全演练」验证模板
- 确认所有模板都能正确识别

## 第四步：开始使用
- 确认模板识别无误后，可关闭安全模式
- 在「指令输入」框中输入需求
- 点击「一键发送到编程工具」
- 使用「一键回传到 ChatGPT」将结果发送给 GPT

## 遇到问题？
1. 点击「系统自检」检查依赖和配置
2. 点击「保存诊断报告」获取详细信息
3. 查看「README_L工具使用说明.txt」了解更多

## 重要提示
- 建议每次大改前点击「创建版本备份」
- 如果改坏了，可以从 backups 文件夹恢复
- 安全模式下只粘贴不发送，更安全
